>Research Project

March.2017----January.2018 Short-term prediction of commercial housing price in Hongshan district based on Web search data （Project Leader）
Dongqi Pu, Shao Dingling, Wu Tingyu, Zeng Jing and Qian Wenying
Abstract： Network retrieval data contains customers' interest, which usually represents their expectations and indicates their next economic behaviours. Therefore, we can use network retrieval traces and words that relate to real estate to short-term predict housing prices. By using the Baidu Search Index (BSI) as sample data, we disposed default values, filtered out zero-variance variables and selected out 23 variables from 60 search keywords. To get the best models, we fitted and forecasted the commercial housing prices in Hongshan district, Wuhan City by using 6 regression models ( Multiple Linear Regression, Classification and Regression Trees, Bagging, Random Forest, Support Vector Machine) in the empirical research section. The results show that the Linear Regression Model and Random Forest Model have the best performance, whose average error rate is -0.11% and 0.13% respectively.
